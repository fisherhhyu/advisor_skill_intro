03 每日运维

📧 每日邮件报告

纯 Shell 实现的日常运维工具，自动接收并生成每日巡检报告

核心特性

  • 多账号/架构图分组管理
  • 本地缓存 + 自动去重（message-id）
  • 环比对比分析风险趋势
  • 自动提取并映射账号名称
  • 纯 Shell 实现，零 Python 依赖

脚本矩阵

  • advisor-report.sh — 主编排脚本
  • advisor-sync.sh — 邮件同步缓存
  • generate-report.sh — 按账号生成报告
  • daily-summary.sh — 聚合每日摘要
  • advisor-compare.sh — 环比对比工具
# 安装依赖 Skills
openclaw skills install himalaya
openclaw skills install summarize

# 生成今日报告
./scripts/advisor-report.sh --today

# 带环比对比
./scripts/advisor-report.sh --today --compare

# 指定账号
./scripts/advisor-report.sh --appid 123456 --today

# 📋 报告输出示例：
🔴 High Risk: 7
🟡 Medium Risk: 13
✅ Healthy: 104
